## 2026-06-08 - Improved File Navigation and Input Clarity
**Learning:** Adding a "My Shared Files" button significantly improves navigation when browsing peer files, as users previously had no obvious way to return to their own shared files list. Replacing cryptic icons like "✓" with descriptive labels like "Set" and adding placeholder examples like "192.168.x.x" reduces cognitive load and prevents user errors.
**Action:** Always provide explicit navigation to return to the 'home' or 'local' state when viewing remote content, and favor descriptive text over icons for action buttons.
